# その他のサービス

## Chat (Workspace)

```bash
# スペースへメッセージ送信
gog chat message spaces/<spaceId>

# ダイレクトメッセージ
gog chat message dm/<userId>
```

## Classroom

```bash
# コース管理
gog classroom courses list

# 課題管理
gog classroom coursework list <courseId>
```

## Keep (Workspace only)

```bash
# ノート管理
gog keep notes list
gog keep notes create
```
